A small command-line tool for making file trees out of blueprints. This is version 0.1.

  • Chapter 1: Top Level

    • Program Control - The top level, which is little more than a demarcation between subcommands.

    • Command Line Arguments - To parse the command line arguments with which inprint was called, and to handle any errors it needs to issue.

  • Chapter 2: Subcommands

    • inprint draw Subcommand - The inprint draw subcommand draws a blueprint from a file tree.

    • inprint build Subcommand - The inprint build subcommand builds a file tree from a blueprint.
